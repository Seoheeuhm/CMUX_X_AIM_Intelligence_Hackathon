Portfolog 문서

Portfolog는 AI가 당신의 경험 자료를 분석하고, 직무에 최적화된 전문 포트폴리오를 자동으로 생성해주는 서비스입니다.

Portfolog란?

기존 이력서·포트폴리오 파일을 올리기만 하면, AI가 프로젝트를 추출하고 성과를 재구성하여 채용 담당자가 바로 읽을 수 있는 수준의 포트폴리오를 HTML로 생성합니다.

개발자, 기획자, 디자이너, 마케터 4개 직군 템플릿을 지원하며, 채용공고 URL을 입력하면 JD 키워드에 맞춰 내용을 재배치합니다.

주요 특징

  • PDF / PPTX / 노션 링크 / 텍스트 직접 입력 지원
  • Claude AI 기반 심층 분석 및 3개 꼬리 질문 인터뷰
  • KPT / STAR 회고 기반 성과 발굴
  • 채용공고 키워드 매핑 (Track B)
  • HTML 생성 후 브라우저에서 즉시 PDF 저장

빠른 시작

Portfolog는 회원가입 없이 바로 사용할 수 있습니다. 아래 단계를 따라 첫 포트폴리오를 3분 안에 만들어보세요.

  1. 1
    portfolog.app 접속 — 상단 '무료로 시작하기' 버튼을 클릭합니다.
  2. 2
    자료 업로드 — 기존 포트폴리오 파일을 끌어다 놓거나, 노션 링크를 붙여넣습니다.
  3. 3
    AI 인터뷰 응답 — AI가 생성한 3개의 꼬리 질문에 간단히 답변합니다.
  4. 4
    직무 선택 — 지원할 직군과 채용공고 URL(선택)을 입력합니다.
  5. 5
    완성 & 저장 — 생성된 포트폴리오를 미리보고 PDF로 저장합니다.
💡 팁: 답변이 구체적일수록 포트폴리오 품질이 높아집니다. 수치(예: MAU 20% 증가, 응답속도 300ms 개선)를 포함해 답변해보세요.

파일 업로드

Portfolog는 다양한 형식의 자료를 지원합니다. 기존 자료가 어떤 형식이든 AI가 분석합니다.

지원 형식

PDF PPTX N Notion 링크 텍스트 입력

업로드 방법

파일 드래그&드롭 또는 클릭하여 선택, 혹은 노션 페이지 공개 URL을 붙여넣으세요. 텍스트 직접 입력 탭에서 경력 내용을 자유롭게 작성할 수도 있습니다.

파일 크기 및 제한

  • 최대 파일 크기: 10MB
  • 복수 파일 업로드 지원 (최대 5개)
  • 노션 링크는 공개(Public) 페이지만 가능
⚠ 주의: 개인정보(주민등록번호, 계좌번호 등)가 포함된 파일은 업로드 전 삭제 후 사용하세요.

AI 인터뷰

업로드한 자료를 분석한 AI가 포트폴리오를 더욱 풍부하게 만들기 위해 3개의 꼬리 질문을 제시합니다.

인터뷰 방식

질문은 업로드한 자료 내용을 기반으로 개인화되어 생성됩니다. 단순한 경험 나열이 아닌, 구체적인 성과와 역할을 이끌어내는 질문입니다.

좋은 답변 작성법

  • 수치를 포함하세요: "성능을 개선했다" → "API 응답속도를 800ms에서 220ms로 단축했다"
  • 내 역할을 명확히 쓰세요: "팀 프로젝트" → "백엔드 리드로 3명 팀을 이끌었다"
  • 결과와 임팩트를 포함하세요: "출시했다" → "출시 후 DAU 15% 성장에 기여했다"
💡 팁: 회고 탭에서 KPT 또는 STAR 방법론으로 먼저 정리해두면 인터뷰 답변이 훨씬 쉬워집니다.

직무 & 트랙 선택

Portfolog는 4개 직군과 2개 생성 트랙을 지원합니다.

지원 직군

  • 개발자 — 기술 스택, 성능 지표, 인프라 경험 중심
  • 기획자 — UX 리서치, 서비스 지표, 기획 방법론 중심
  • 디자이너 — 툴 역량, 브랜딩, UX 개선 사례 중심
  • 마케터 — 퍼포먼스 지표, 채널 전략, 캠페인 성과 중심

트랙 선택

Track A (기본) — 업로드 자료와 인터뷰 답변만으로 범용 포트폴리오를 생성합니다.

Track B (채용공고 맞춤) — 채용공고 URL을 입력하면 JD의 핵심 키워드를 분석해 포트폴리오 내용을 재구성합니다. 서류 통과율을 높이는 데 최적화되어 있습니다.

💡 팁: Track B 사용 시 지원하려는 채용공고의 URL을 직접 붙여넣으면 가장 정확한 키워드 매핑이 이뤄집니다.

포트폴리오 완성

모든 정보를 입력하면 AI가 자동으로 포트폴리오 HTML을 생성합니다. 완성까지 약 1~2분이 소요됩니다.

생성 섹션

기본적으로 아래 8개 섹션이 자동 생성됩니다:

  • 자기소개 및 핵심 역량 요약
  • 주요 프로젝트 (최대 5개)
  • 기술 스택 / 사용 도구
  • 업무 경험 및 이력
  • 성과 및 수치 지표
  • 교육 및 자격증
  • 회고 및 성장 서사 (Track B 포함)
  • 앞으로의 목표 (Track B 포함)

PDF 저장

생성된 포트폴리오 페이지에서 브라우저의 인쇄(Ctrl+P / Cmd+P) → PDF로 저장을 선택하세요. A4 세로 형식으로 최적화되어 있습니다.

⚠ 주의: 생성된 HTML은 현재 세션에만 유지됩니다. Pro 요금제에서 클라우드 저장이 지원됩니다.

Free vs Pro 요금제

Portfolog는 무료로 시작할 수 있으며, 더 많은 생성 횟수와 클라우드 저장이 필요할 때 Pro로 업그레이드하세요.

Free 플랜

  • 계정당 포트폴리오 생성 3회
  • 4종 직무 템플릿 전체 이용
  • PDF 다운로드 가능
  • 클라우드 저장 미지원
  • Track B (채용공고 맞춤) 미지원

Pro 플랜 (₩9,900/월)

  • 월 10회 포트폴리오 생성
  • 3GB 클라우드 프로젝트 파일 저장
  • Track B 채용공고 맞춤 최적화 완전 지원
  • 4종 직무 템플릿 전체 이용
  • PDF 다운로드 가능
💡 팁: 취업 준비 기간에만 Pro를 사용하고, 취업 후 Free로 다운그레이드할 수 있습니다.
